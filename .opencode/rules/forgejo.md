# GitHub

This repo is hosted at `github.com/jwilger/eddy`. Use `gh` for issues, pull requests, review comments, and workflow checks.

Prefer ordinary `gh` and GitHub Actions workflows over the old Forgejo-specific tools. If `gh` lacks a required operation, state the gap and use the narrow GitHub REST or GraphQL API endpoint for that operation only.

Inline review feedback must be answered on the inline thread first. Use GitHub review-thread replies for inline comments, not a top-level PR comment.
