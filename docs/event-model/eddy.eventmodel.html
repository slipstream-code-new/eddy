Eddy Course Session Event Model

Overview

Machine-readable Event Modeling source for Eddy's EventCore-backed server behavior. Streams are EventCore consistency boundaries, not traditional aggregates.

Streams

session

id_patternsession-{session_id}
purposeConversation, lesson progress, assistant-turn lifecycle, tool-call lifecycle, dev-server reload lifecycle, and protocol compatibility for one course session.

artifact

id_patternartifact-{artifact_id}
purposeEddy's observed history of an artifact, usually a file, that may also change outside Eddy's control.
