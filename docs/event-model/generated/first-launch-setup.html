First launch setup

Overview

A new learner launches Eddy in the terminal, provides a friendly display name, configures OpenAI access, verifies the connection, and is taken to Lesson 1.

Streams

learner_configuration

id_patternlearner-configuration-{learner_id}
purposeStores the learner-facing setup choices Eddy needs after first launch.

lesson_workspace

id_patternlesson-workspace-{repository_id}
purposeTracks preparation of the local repository workspace for the active lesson.
